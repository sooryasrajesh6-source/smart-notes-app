# 📝 Smart Notes App

A modern and responsive **Smart Notes App** built with **HTML, CSS, and JavaScript**. It allows users to create, search, and delete notes while automatically saving them in the browser using Local Storage.

---

## ✨ Features

- ➕ Add new notes
- 🔍 Search notes instantly
- 🗑️ Delete notes
- 💾 Automatically saves notes using Local Storage
- 📱 Responsive design
- 🎨 Clean and modern user interface

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/sooryasrajesh6-source/smart-notes-app.git
```

### 2. Open the project

Open the project folder and launch `index.html` in your web browser.

---

## 💡 How to Use

1. Enter your note in the input box.
2. Click **Add Note**.
3. View all saved notes.
4. Search notes using the search bar.
5. Delete notes when they are no longer needed.
